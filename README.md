# Retail Profitability Analysis (Marimo Portfolio)

This is my individual project for the Data Science and AI Tools module.

## Overview

This project presents an interactive retail profitability model.  
Users can adjust cost, markup and logistics to see how profit changes.

## Features

- Interactive sliders (COGS, markup, logistics)
- Reactive calculations (revenue, profit, margin)
- Data analysis using pandas
- Visualisations using Plotly
- Built using marimo

## How to run

```bash
marimo edit profitability.py
